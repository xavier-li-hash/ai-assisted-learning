## 如何使用 Claude Code 辅助学习

每次开始学习，告诉 Claude：

```
"今天学 [工程名] [阶段] [主题]，
背景参考 [工程目录]/README.md"
```

例如：
- `"今天学Rust的生命周期，背景参考 rust/README.md"`
- `"帮我做今天的Flink练习，Week2 Day3"`
- `"批改我的雅思写作Week1 Day2"`
