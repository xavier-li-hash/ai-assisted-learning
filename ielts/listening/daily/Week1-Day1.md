## Day 1–2：预读技巧练习

**预读练习（做题前先做这个）：**

在听音频之前，花30秒看以下"题目"，预判答案类型：

```
题目样本（模拟Part 1格式）：

Name of hotel:     _______________     → 预判：专有名词
Check-in date:     _______________     → 预判：日期格式（如 15th March）
Number of nights:  _______________     → 预判：数字（1–30）
Room type:         _______________     → 预判：词组（single/double/suite）
Price per night:   $______________     → 预判：数字，注意"包不包含早餐"
Special request:   _______________     → 预判：形容词/名词（quiet room / high floor）
Phone number:      _______________     → 预判：一串数字，注意英式发音（0=oh）
Email address:     _______________     → 预判：字母+符号，会拼写
```

**练习：** 用上面的格式，在听音频前写下你对每题"答案会是什么类型"的预判。

## 常见陷阱练习（不需要音频）

判断下列哪个是正确答案：

**陷阱1：数字**
> 说话人说："It's fifty... no wait, fifteen dollars."
> 答案应该是：___fifteen___

**陷阱2：拼写**
> 说话人拼出："B-A-R-T-O-N"
> 答案是：___BARTON___

**陷阱3：改口**
> 说话人说："The event is on Saturday... actually, it's been moved to Sunday."
> 答案是：__Sunday____

（答案：15 / Barton / Sunday）

## 错误原因分类

在对应格子写：
- `速度` — 语速太快跟不上
- `连读` — 两个词连在一起没听清
- `生词` — 不认识这个词
- `数字` — 数字听错了
- `拼写` — 字母组合拼错了
- `改口` — 没注意到说话人改了答案
- `预读` — 没来得及看题

## Day 5：精听跟读

选出本周错题最多的那道（或整个Part 1），做精听：

1. 不看答案，再听一遍
2. 对照原文脚本，找到你漏掉的地方
3. 大声跟读那一段，模仿语调
4. 再听一遍（不看脚本），检验是否能听懂了
