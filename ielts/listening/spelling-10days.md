# 听力拼写专项：考前10天冲刺（2026-04-23 起）

> 每天15-20词，按题材分组。格式：看中文→默写英文→对答案
> **原则**：不会写的词，用记忆口诀拆分，写3遍，第二天必须复查

---

## 使用方法

1. 每天只做当天的题材
2. 看中文提示，在空格处写出英文单词
3. 完成后翻到文末对答案
4. 不会的词：旁边标 ★，写3遍，明天复查
5. Day 10 是全部高频词的集中默写

**今日记忆口诀：**
- `balcony` → bal-**con**-y（阳台有个"con"骗人，记住中间是 con）

- `wardrobe` → ward（病房/守护）+ robe（长袍）→ 守护长袍的柜子
- `accommodation` → **ac-com-mo-da-tion**（两个 c，两个 m）→ 口诀：**"A Cat Can Make More Mess"**（首字母）
- `corridor` → cor-ri-**dor**（双 r，结尾 dor 不是 der）
- `microwave` → micro（微小）+ wave（波）→ 微波
