### 三、针对你的核心策略：遇到生词时怎么办

---

#### 策略1：建立"核心词应急推断法"

当你遇到不认识的核心词（比如arson），**不要放弃，做三步推断**：

```
步骤1：看词本身有没有熟悉的词根
  arson → 看不出来，跳过

步骤2：看上下文给出了哪些线索
  - "fire was deliberately set"（故意放火）
  - "course in fire science"（消防科学）
  - "Arson for Profit"（为了利润的 arson）
  - "a firefighter to torch a building"（消防员去烧楼）
  → 推断：arson = 故意纵火（尤其是为了钱）

步骤3：用你的推断读下去，不需要100%准确
  "大概是故意放火骗保险之类的犯罪" → 已经够用了
```

这个技能的核心是**容忍模糊**：不认识一个词，不等于放弃这段。70%的理解就够判题。

---

#### 策略2：标题匹配用"主旨句提炼法"

**不要用"关键词联想"** → 容易被表面词汇带偏（比如marketing → financial）

正确方法：读完每段，用一句话概括作者的**核心论点**，然后去匹配标题。

| 段落 | 作者在说什么（核心论点） | 匹配标题 |
|------|------------------------|----------|
| B | 这门课也可能被坏人利用 | H（吸引错误学生） |
| C | "Principles"这个词本身有歧义 | B（课名双重含义） |
| D | 引入康德的哲学框架来分析 | D（在意外场景运用理论） |
| F | 同知识用于不同目的就叫不同名字 | G（不同结果不同名称） |

**练习方法**：做完标题匹配后，反过来验证——用你选的标题去读段落，看它是不是段落的主要内容，而不只是某句话。

---

#### 策略3：TFNG 的三格判断框架

遇到TFNG，用这个顺序：

```
第一步：在原文定位这个话题（能不能找到相关句子？）
  → 找不到任何相关内容 → NOT GIVEN

第二步：找到了，看原文支持还是反对题目的说法
  → 支持 → YES
  → 反对 → NO

注意陷阱：
  ① "目的"≠"效果"（Q38）
  ② 否定句要特别敏感（Q39 的 "My point is NOT to criticize"）
  ③ "模糊/笼统"≠"没有"（Q40 的 generally vs detailed）
```

---

#### 策略4：答案管理（针对标题匹配）

每次选了一个标题后，**立刻在选项旁画叉**，确保不会重复使用。

```
选项列表示例：
A. ...
B. ... ✓ → 用于C段
C. ...
D. ... ✓ → 用于D段
...
H. ... ✓ → 用于B段（已用，不能再填F段）
```

你Q32把H填进F段，根本原因是没有严格管理已用选项。

### 四、本篇文章对你的额外价值

这篇文章的**论证结构**是雅思抽象议论文的典型模板，你以后会遇到很多这种类型：

> **核心论点** → **举具体例子（Arson课）** → **引入哲学框架（Kant）** → **用框架解释例子** → **得出结论**

当你发现文章在讲一个抽象论点（什么是知识的目的？），**不要去找具体事实**，而是追踪作者在**一步一步论证什么**。这类文章的标题匹配，匹配的是论证步骤，不是信息点。
