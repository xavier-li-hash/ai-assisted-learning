## 二、信息匹配——隐晦信息定位技巧

### 核心技巧1：找"行为主体 + 动作"而不仅找名词

题目通常描述"某人/某物做了某事"。定位时不只找名词，要同时确认动词意义。

| 题目表述 | 原文可能的对应方式 |
|---------|-----------------|
| researchers discovered... | scientists found / identified / revealed... |
| governments failed to... | authorities did not / neglected to / were unable to... |
| the technology was initially opposed | early resistance to the method / critics argued against |
| higher costs led to... | the expense resulted in / financial burden caused... |

### 核心技巧2：警惕"相似段落"陷阱

当多个段落都讨论同一主题（如"污染"），用以下方式区分：
- **A段** 描述问题现状 → 题目问"描述了某现象"
- **B段** 分析原因 → 题目问"解释了某原因"
- **C段** 提出解决方案 → 题目问"建议了某方法"

**做法：** 确认段落的"功能角色"，不只是"主题匹配"。

### 核心技巧3：隐晦信息的4种藏法

```
1. 举例中藏信息    "...such as the case in Norway, where subsidies tripled"
                   → 题目问 "financial support increased significantly in one country"

2. 让步从句中藏信息  "Although widely accepted, the model has been challenged..."
                   → 题目问 "doubts have been raised about a commonly used approach"

3. 比较结构中藏信息  "Region A showed twice the rate of decline seen in Region B"
                   → 题目问 "one region experienced a more rapid decrease"

4. 否定句中藏信息   "The study found no evidence that temperature affected results"
                   → 题目问 "temperature was shown to have no impact"
```

## 七、错题深度分析框架


做完后，对每道错题回答这3个问题：

```
Q1：我的定位词是什么？原文中对应的词/句是什么？
     → 找出 paraphrase 的落差

Q2：我为什么选错了段落/选了错误答案？
     → 是被表面词汇误导？还是忽略了动作主体？

Q3：这道题的隐晦信息用了上面哪种"藏法"？
     → 举例藏法 / 让步从句 / 比较结构 / 否定句
```
