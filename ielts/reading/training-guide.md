## True/False/Not Given 策略

1. 先读题，划关键词
2. 在文章中定位关键词所在段落
3. 逐字比较：
   - 信息一致 → True
   - 信息矛盾 → False
   - 文章没提到 → Not Given（不是"感觉上没有"，是"真的没有"）
4. **最难区分**：False vs Not Given
   - False = 文章说了相反的
   - NG = 文章根本没说这件事

---

## Matching Headings 策略

1. 先读所有Heading选项，理解大意
2. 每段只读第1句和最后1句（段落主旨往往在这）
3. 排除法：已用的Heading划掉
4. 警惕干扰项：文中出现的词不一定是正确Heading

## 关键技能：同义替换识别

雅思阅读答案不直接抄原文，常见替换：
| 原文 | 题目 |
|------|------|
| children | young people / kids |
| increased | rose / grew |
| difficult | challenging / hard |
| government | authorities / policymakers |

每次做完题，把你发现的同义替换写下来，积累词库。
