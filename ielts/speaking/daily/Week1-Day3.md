## 热身（2分钟）

大声朗读以下句子，注意语调起伏（每句最后的关键词往下降调）：
```
I tend to be more productive in the *mornings*, so I try to start work *early*.
Most of the time I commute by *train*, which actually gives me time to *read*.
I've started cooking at home more *often* recently, mainly to *save money*.
My evenings are usually quite *low-key* — just dinner, maybe a walk, then *bed*.
So overall, I'd say my daily routine is pretty *structured*, which suits me *well*.
```

## 逻辑链条挑战（5分钟）

> **今天不给你填空提示——只给起始句，你自己展开完整的4步链条。**
>
> 规则：不允许只说"I like it" / "it is good" — 必须有**具体时间、地点、数字、或真实细节**

**挑战1：**
```
起始句：I'm more of a morning person.
→ 你来说4句（原因 → 例子 → 具体细节 → 影响/感受）
```

**挑战2：**
```
起始句：My smartphone plays a big role in my daily life.
→ 你来说4句（不能只说"it's very useful"，说具体怎么用、何时用）
```

**挑战3（对比型）：**
```
起始句：Cooking at home and eating out feel very different to me.
→ 你来说4句（两面各说2句，用 while / whereas / on the other hand 连接）
```

> 完成后问自己：每句里有没有至少一个**具体词**（人名/地名/数字/时间）？

## 今日升级目标

前两天你练的是2–3句。今天开始练4句：

```
句1：直接回答（观点）
句2：理由
句3：具体例子或细节
句4：补充说明或总结
```

示例：
> 问：Do you prefer cooking at home or eating out?
>
> 句1：I personally prefer eating out.
> 句2：It saves a lot of time and effort after a busy workday.
> 句3：For example, there's a small restaurant near my home that I go to at least twice a week.
> 句4：But I do enjoy cooking on weekends when I have more time.
