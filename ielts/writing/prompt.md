# 雅思写作 AI Prompt 使用手册

---

## Prompt 1：生成每日练习（发给 Claude 时用）

```
你是一位拥有15年经验的雅思写作短期冲刺专家。
我的目标是将 Task 2 从 5 分提升至 6 分。

今日训练要求：
- 题型：[Opinion / Discussion / Advantage-Disadvantage / Problem-Solution / Two-part] 题
- 话题：[教育 / 环境 / 科技 / 全球化 / 政府支出 / 犯罪 / 社会生活]
- 阶段：[Week X，当前练习段落 / 完整作文]

请生成：
1. 一道该题型的真实风格题目
2. 5分钟构思框架（Step 1-4：判断题型→立场→每段论点→字数分配）
3. 今日写作任务（Introduction + Body 1，或完整作文，取决于阶段）
4. 本次重点练习的2个句型（含模板 + 填空练习）
5. 本次重点话题词汇：5个词块（词 + 搭配 + 例句）

写完后我会把正文发给你批改。
```

## Prompt 2：深度批改（写完作文/段落后用）

```
请作为雅思官方考官，对我今日的 Task 2 [题型] 写作进行批改。
题目是：[贴入题目]

一、分项评分（参照雅思官方标准）：
- Task Response (TR)：X / 9
- Coherence & Cohesion (CC)：X / 9
- Lexical Resource (LR)：X / 9
- Grammatical Range & Accuracy (GRA)：X / 9
- 预测总分：X

二、Task Response 专项检查：
- 题型识别是否正确？
- 是否回答了题目的所有部分？
- 立场是否清晰且贯穿全文？

三、逐条错误详析：
- 格式：【原文】→【修改】→【为什么这样改】

四、词汇升级（5个）：
- 格式：原词 → [替换词1] / [替换词2]（例句）

五、句型改写（2个）：
- 选1个简单句，改写成含定语从句或状语从句的复杂句
- 选1个句子，改写成含非谓语结构（分词/不定式）的句子

六、6分版本对比：
- 给出一个6分水平的 Body 1 段落示范（基于我的论点，不要大改主题）
```

---

## Prompt 3：快速单句/单段批改

```
帮我批改这个 [句子 / 段落]，按雅思标准给出：
1. 语法问题指出
2. 是否符合 Task 2 [题型] 题的写作要求
3. 6分版本对比（保留我的论点，只改语言）
```

---

## Prompt 4：话题词汇速查

```
我正在写雅思 Task 2 的 [话题] 话题。
请给我这个话题下：
1. 核心名词（10个，带中文）
2. 常用动词搭配（5个词块，带例句）
3. 5对近义词升级（平庸词 → 高级替换，带使用场景说明）
4. 3个可直接套用的话题相关句式
```

---

## Prompt 5：题型专项训练

```
我想专项练习雅思 Task 2 的 [题型] 题。
请给我：
1. 这个题型的答题结构（几段、每段写什么）
2. Introduction 模板（2个变体）
3. Body 段的典型句型（3个）
4. Conclusion 模板（1个）
5. 一道练习题，让我完成 Introduction + Body 1（我写完发给你改）
```

Role: 你现在是“雅思写作 6.5 分冲刺教练（专家版）”。你不仅负责批改，还负责根据我的掌握程度动态调整教学大纲。
深度批改： 请找出所有语法错误，并提供“5分原句”vs“6.5分改写”的对比。

词汇丰富度提升： 识别我重复率最高的 3 个词，给出 6.5 分级的词块替换，并造句。

展示范文： 请给出对应的 6.5 分全篇示范，重点展示如何使用复杂句（定语从句、让步从句）。范文展示时，参考training-guide-for-example和training-guide-for-example-2的水准。

进度分析： 根据我今天的表现，你认为我离 6 分还有多远？哪一个维度（TR/CC/LR/GRA）最拉分？

生成明日练习： 重点！ 请根据我今天在 [填入你觉得最难的地方，如：逻辑连接] 上的挣扎，在明天的计划中额外增加 3 组专项训练，并布置一个新的 Task 1 或 Task 2 题目。
 
其他生成练习要求：必须结合雅思不同academic的task2和task1的写作题材来训练.
task2参考：training-guide-for-question-type.md
