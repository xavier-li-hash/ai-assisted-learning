# 写作 Week 1 - Day 4："具体化"专项训练

> 目标：把模糊句改成具体句——这是雅思得7分的核心能力
> 核心技巧：把"变好了"变成"具体是怎么好的"

---

## 今日语法焦点

**模糊句 vs 具体句对比：**

| 模糊（5.5分） | 具体（7分） |
|-------------|------------|
| Technology helps people. | Technology allows people to access information instantly through smartphones. |
| Exercise is good for health. | Regular jogging for 30 minutes a day strengthens the heart and reduces stress. |
| Education is important. | Education equips students with critical thinking skills needed for the job market. |

**具体化三问：**
1. 谁在做？（who？）
2. 做什么具体的事？（what exactly？）
3. 带来什么具体结果？（specific outcome？）

---

## 今日练习（5句）

**1.** 把这句话变具体：
```
原句：Social media is helpful for young people.
你的具体句：Social media helps young people to ______, especially by ______.
```

**2.** 把这句话变具体：
```
原句：The environment is getting worse.
你的具体句：______ （用具体数据/现象/地点来描述）
```

**3.** 把这句话变具体：
```
原句：Working too much is bad.
你的具体句：______ （谁？具体怎么bad？）
```

**4.** 主动造句（不给框架）：
```
写一句关于"读书"的具体句，用到：students / by reading / which helps them
```

**5.**（挑战）
```
写两句话：
第1句：一个现象（具体）
第2句：这个现象的结果（具体）
话题：手机使用
```

## Task 1 专项：数据对比描述句训练

> **考试场景：** Task 1 body 段常需要跨类别或跨时间段比较数据——哪个更高？差距多少？谁排第一？
> **核心原则：** 用比较句型替代连续陈述，让信息更紧凑。

**对比核心句型：**

```
[A] was significantly higher than [B], at X% compared to Y%.
While [A] accounted for X%, [B] represented only Y%.
[A] and [B] showed contrasting trends: the former rose while the latter fell.
[Category] had the highest/lowest figure, at X%, followed by [Category 2] at Y%.
```

**今日练习（4句）：**

**C-1.** 用 `compared to` 写一句对比句：
```
图表信息：2020年，男性就业率为75%，女性为58%。
你的句子：The employment rate for men stood at ______, compared to ______ for women.
```

**C-2.** 用 `while` 写一句两者趋势对比句：
```
图表信息：1990–2020年，A国能源消耗持续上升，B国持续下降。
你的句子：While [Country A]'s energy consumption ______, that of [Country B] ______ over the same period.
```

**C-3.** 用排名句式描述最高/最低：
```
图表信息：三种交通方式使用率：汽车55%，公交25%，自行车20%。
你的句子：______ was the most popular mode of transport, at ____%, followed by ______ at ____% and ______ at ____%.
```

**C-4.** 用 `the former... the latter` 对比两组数据：
```
图表信息：私立学校和公立学校的学生人数：私立从2000年的10万增至30万，公立从80万降至60万。
你的句子：Both private and public school enrolment changed considerably: the former ______, while the latter ______.
```

**自查：**
- [ ] `compared to` 前后两个对象是同类可比的吗？
- [ ] `the former / the latter` 对应的是前句提到的哪两个？
- [ ] 有没有同时写了数字又写了趋势（不要重复信息）？
