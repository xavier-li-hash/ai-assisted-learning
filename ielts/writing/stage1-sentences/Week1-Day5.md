## Task 1 专项：流程图 & 地图描述句训练

> **考试场景：** Task 1 偶尔出现流程图（process diagram）和地图（map），描述方式与数据图完全不同——不谈变化趋势，而是描述步骤顺序或地理位置变化。

---

### 流程图描述句型

**核心逻辑：** 用顺序连接词（First → Then → Next → After that → Finally）描述每个步骤，动词用被动语态为主。

```
First, [raw material] is [processed/treated/transported] to [location/stage].
After this, [the product] is [verb-ed] in order to [purpose].
Finally, [the finished product] is [distributed/packaged/stored].
```

**P-1.** 用被动语态描述流程步骤：
```
流程：原材料 → 被运送到工厂 → 被清洗 → 被切割 → 被包装 → 发送给消费者
写出前3个步骤的句子：
First, the raw materials ______.
They are then ______ before being ______.
```

**P-2.** 连接两个步骤，用 `Once... it is then...` 句型：
```
流程步骤：水先被过滤，过滤完后才加热。
你的句子：Once the water ______, it is then ______.
```

---

### 地图描述句型

**核心逻辑：** 描述地点变化（新增/拆除/移动）+ 位置关系（方位词）。不写趋势，写位置和变化。

**位置描述词汇：**
```
to the north/south/east/west of
in the centre/on the outskirts of
adjacent to / next to / opposite
where [X] once stood, there is now [Y]
```

**变化描述句型：**
```
The [building/road/area] has been replaced by ______.
A new [facility] has been constructed [to the east of / next to] ______.
What was formerly [X] is now [Y].
The [area] to the north of the river has been developed into ______.
```

**M-1.** 描述一处"新增"的变化：
```
地图信息：2020年，学校东边原来是空地，现在建了一个停车场。
你的句子：To the east of the school, ______ has been replaced by ______.
或：A ______ has been constructed ______.
```

**M-2.** 用 `What was formerly... is now...` 描述用途改变：
```
地图信息：原来的工厂区现在变成了住宅区。
你的句子：What was formerly ______ is now ______.
```

**M-3.** 综合写2句话描述下面的地图变化：
```
假设地图：2000年地图中，河流南边是农田，北边是工厂。
        2020年地图中，农田变成了购物中心，工厂被拆除，北边建了公园。
你的2句话：______
```
