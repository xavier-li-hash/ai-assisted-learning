### 今日新句型：因果句 & 目的句

```
【因果句——结果在后】
X leads to Y.
X results in Y.
X has caused Y to + V.
X contributes to Y by doing Z.

【因果句——原因在后】
Y is largely due to X.
Y results from X.
As a result of X, Y has become more...
Due to X, Y tends to...

【目的句】
In order to X, governments should...
So as to achieve X, it is essential to...
X is introduced with the aim of Y-ing...
...in order that X can be achieved.
```

## Task 1 专项：流程图（Process Diagram）描述训练

> **考试场景：** 流程图描述一个制造/循环过程，重点用被动语态 + 顺序词，按步骤说明每个阶段。

---

### 流程图核心句型

```
【顺序词】
First / To begin with / Initially
Then / Next / Subsequently / After that
Finally / Lastly / In the final stage

【被动语态描述步骤】
[材料] is collected / processed / transported / converted / treated.
Once [X] is completed, [Y] is then [verb-ed].
After [X] has been [verb-ed], it is then [verb-ed].
[原料] undergoes [过程名], during which [something happens].

【整体概述】
Overall, the process involves X main stages, beginning with [A] and ending with [B].
The diagram illustrates a [linear/cyclical] process in which [主要描述].
```

### 今日练习（5句）

> 参考以下模拟流程图：**纸张回收流程（5步）**

```

Step 2: 分类处理（Paper is sorted by type and quality）
Step 3: 碎纸制浆（Paper is shredded and mixed with water to form pulp）
Step 4: 清洁过滤（Pulp is cleaned and filtered to remove impurities）
```

## 语法焦点：Week 3 新句型速查

```
【因果句】
X leads to Y.                           → X 导致 Y
X results in Y.                         → X 导致 Y（结果较严重）
Y is largely due to X.                  → Y 主要由 X 引起
As a result of X, Y has increased.      → 由于 X，Y 增加了
X contributes to Y by doing Z.          → X 通过做 Z 促成了 Y

【目的句】
In order to X, S + V.                   → 为了 X，……
S + V + in order to X.                  → ……以达到 X
...with the aim of V-ing...             → ……以……为目的
...so as to + V...                      → 以便……

【常见搭配】
invest in → 在……方面投资
contribute to → 对……有贡献
lead to / result in / give rise to → 导致
be attributed to → 归因于
```

## Day 7 复习答案汇总

> 做完复习部分后再看。

### 复习 A 答案

**RA1.** "Although automation can replace some workers, ~~but~~ it **can still** create new job opportunities."
（Although 和 but 不能同时使用；副词 still 放在情态动词 can 之后）

**RA2.** "Despite its high cost, renewable energy **is widely used** in developed countries, which helps reduce carbon emissions."
（"widely used" 缺谓语 is；"which" 前加逗号构成非限制性从句）

**RA3.** "Students in urban areas are significantly **easier** to access high-quality education than **those** who live in rural areas."
（"more easier" → "easier"；"than who" → "than those who"）
