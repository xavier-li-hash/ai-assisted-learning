# 写作训练说明

## 三阶段路径

```
第1阶段（Week 1–3）：句子训练
  → 半成品补全 → 主动造句 → 句型变换

第2阶段（Week 4–5）：段落训练
  → 4–5句话的段落，有主题句+支撑+结尾

第3阶段（Week 6–8）：完整作文
  → Task 2 议论文 → Task 1 图表
```

## 每次批改时，告诉Claude：

"帮我批改这句/段/篇写作，按雅思标准给出：
1. 语法问题指出
2. 逻辑是否清晰
3. 6.5分版本 vs 7分版本的对比"

## 常用句型模板（第1阶段用）

### Task 2 议论文句型

**因果句：**
- `X can improve Y by doing Z, which helps...`
- `Because X, people tend to...`
- `As a result of X, Y becomes...`

**让步句：**
- `Although X, it is still important to...`
- `While some argue that X, others believe...`

**对比句：**
- `Unlike X, Y tends to...`
- `Compared to X, Y is more effective because...`

**目的句：**
- `In order to X, people should...`
- `So that X can be achieved, it is essential to...`

### Task 1 图表/流程/地图句型

**Overview 总结句（第2段，不写数字）：**
- `Overall, it is clear that X, while Y.`
- `In general, X showed an upward trend, whereas Y remained relatively stable.`
- `The most striking feature is that X, with Y being the exception.`

**数据变化趋势句（线图/柱图 body 段）：**
- `[主语] rose/fell/climbed sharply to [数字] in [年份].`
- `There was a dramatic increase in [主语], reaching [数字] by [年份].`
- `[主语] peaked at [数字] in [年份], before declining in subsequent years.`
- `[主语] fluctuated between [数字] and [数字] before levelling off at [数字].`

**数据对比句：**
- `[A] stood at X%, compared to Y% for [B].`
- `While [A] accounted for X%, [B] represented only Y%.`
- `[A] had the highest figure at X%, followed by [B] at Y%.`
- `Both A and B changed: the former rose while the latter fell.`

**流程图句（被动语态 + 顺序词）：**
- `First, [材料] is [processed/transported/treated].`
- `Once [X] is completed, [Y] is then [verb-ed].`
- `After this, [主语] is [verb-ed] in order to [purpose].`
- `Finally, [成品] is [distributed/packaged/stored].`

**地图描述句（位置 + 变化）：**
- `To the north/south/east/west of [X], [Y] has been constructed.`
- `What was formerly [X] is now [Y].`
- `[Building] has been replaced by [new facility].`
- `Where [X] once stood, there is now [Y].`
