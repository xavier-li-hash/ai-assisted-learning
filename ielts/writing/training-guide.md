## 一、Task 2 五大题型答题框架

> 每种题型有**不同的段落组织逻辑**，这是 5→6 最关键的跨越。

---

### 题型 1：Opinion Essay（同意/不同意）

**题目特征：** "Do you agree or disagree?" / "To what extent do you agree?"

**结构：**
```
Introduction：背景句 + 立场（明确同意/不同意）
Body 1：      支持自己立场的理由1（PEEL）
Body 2：      支持自己立场的理由2，或让步+反驳
Conclusion：  重申立场
```

**Introduction 模板：**
```
[改写题目背景]. While some people believe [对立观点],
I would argue that [你的立场], because [核心理由简述].
```

**让步+反驳模板（Body 2 变体）：**
```
Admittedly, [对立观点有一定道理].
However, [但为何你的立场更合理].
```

---

### 题型 2：Discussion Essay（讨论两方观点）

**题目特征：** "Discuss both views and give your own opinion."

**结构：**
```
Introduction：背景句 + 说明将讨论两方 + 给出自己倾向
Body 1：      一方观点的理由（PEEL）
Body 2：      另一方观点的理由（PEEL）
Conclusion：  总结两方 + 重申个人倾向
```

**Introduction 模板：**
```
[改写题目]. Some people think [观点A], while others argue [观点B].
This essay will discuss both perspectives, and I believe [你的立场].
```

**⚠️ 常见失分点：**
- Body 1 和 Body 2 必须都有论据，不能只讲一方
- Conclusion 里必须再次给出个人意见

---

### 题型 3：Advantage / Disadvantage Essay

**题目特征：** "What are the advantages and disadvantages?" / "Do the advantages outweigh the disadvantages?"

**结构（A）：** 纯列举
```
Introduction：背景 + 说明将分析利弊
Body 1：      优点（2个支撑点）
Body 2：      缺点（2个支撑点）
Conclusion：  权衡总结（说清楚利大于弊还是相反）
```

**结构（B）：** outweigh 题（必须表态）
```
Introduction：背景 + 明确表态（利大于弊/弊大于利）
Body 1：      权重较大的那一方（2个论点）
Body 2：      权重较小的那一方（1个论点 + 反驳）
Conclusion：  重申判断
```

---

### 题型 4：Problem / Solution Essay

**题目特征：** "What are the causes/problems? What solutions can you suggest?"

**结构：**
```
Introduction：背景 + 说明将分析问题和解决方案
Body 1：      问题/原因（2个，每个配解释）
Body 2：      解决方案（对应上面的问题，2个）
Conclusion：  总结
```

**因果链模板：**
```
One major cause of [问题] is [原因].
This leads to [结果], which means [影响].
To address this, [解决方案] could be implemented.
```

---

### 题型 5：Two-Part Question

**题目特征：** "Why is X happening? Do you think this is positive or negative?"

**结构：**
```
Introduction：背景 + 说明将回答两个问题
Body 1：      回答第一个问题（Why/What）
Body 2：      回答第二个问题（Is it good/bad? Opinion?）
Conclusion：  总结两个答案
```

**⚠️ 关键：两个问题都必须回答，且权重相当，不能只写其中一个。**

## 二、Task 2 七大话题词汇库

> 每个话题：核心名词 → 动词搭配 → 近义词对 → 常用表达

---

| 类型 | 词汇 |
|------|------|
| 核心名词 | curriculum, tuition fees, academic performance, vocational training, critical thinking |

| 近义词对 | teach → educate / instruct；learn → acquire / absorb |
| 常用表达 | equip students with practical skills；bridge the gap between education and employment |

## 三、近义词升级策略（LR 得分关键）

> 6分要求：不重复使用同一个词；用词准确（不求华丽）

### 高频平庸词 → 升级替换

| 平庸词 | 升级选项 | 使用场景 |
|--------|----------|----------|
| **important** | crucial, significant, vital, essential | crucial 强调紧迫；vital 强调不可缺少 |
| **show** | demonstrate, indicate, reveal, suggest | reveal 含有"揭示出人意料"之意 |
| **problem** | issue, challenge, concern, drawback | challenge 较中性；drawback 指缺点 |
| **help** | assist, support, enable, facilitate | facilitate 指"使...更容易" |
| **increase** | rise, grow, expand, surge, soar | surge/soar 指急剧增加 |
| **decrease** | fall, drop, decline, reduce, plummet | plummet 指大幅下跌 |
| **think** | argue, believe, maintain, suggest | argue 用于表达有论据的观点 |
| **because** | since, as, given that, due to the fact that | given that 更正式 |
| **but** | however, nevertheless, nonetheless, yet | nevertheless 表强烈转折 |
| **also** | furthermore, moreover, in addition, besides | moreover 语气比 also 更强 |

## 四、大作文构思框架（5分钟 Brainstorm）

> 考场上先花5分钟构思，这是拿到 6 分 Task Response 的前提。

### 步骤

```
Step 1（30秒）：判断题型
  → 看题目是 Opinion / Discussion / A&D / P-S / Two-part？

Step 2（1分钟）：确定立场
  → Opinion 题：我同意还是不同意？（选容易写的那方）
  → Discussion 题：两方各有什么理由？我倾向哪方？

Step 3（2分钟）：每段想1-2个论点
  → 用"Why? So what? For example?"三问法展开每个论点
  → 论点要具体，不要只写"it is good for society"

Step 4（1分钟）：框架确认
  → 4段：Introduction + Body 1 + Body 2 + Conclusion
  → 每个 Body 段：80-100 字（4-5句）
  → 总字数：≥ 250 字
```

### "Why? So what? For example?" 展开法

```
论点：Sports facilities improve public health.

Why（为什么）：Because they provide accessible space for exercise,
               encouraging people to be physically active.

So what（有什么影响）：This leads to lower rates of obesity and
                      lifestyle-related diseases.

For example（举例）：In Singapore, the government built community
                    sports centres in every neighbourhood, and
                    obesity rates fell by 10% over a decade.
```

## 五、段落写作公式（各题型差异版）

### 通用 PEEL 结构

```
① Point（论点句）：   直接陈述本段核心观点（1句）
② Explain（解释句）： This is because... / This means that...（1-2句）
③ Example（例子句）： For example, / For instance,（1句）
④ Link（小结句）：    Therefore, / As a result, / This suggests that...（1句）
```

### Opinion 题 Body 段（支持自己立场）

```
[论点]. This is because [原因], which [影响].
For instance, [具体例子].
Therefore, [重申论点对整体立场的意义].
```

### Discussion 题 Body 段（介绍一方观点）

```
Those who support [观点A] argue that [理由].
They believe that [进一步解释].
For example, [例子].
This perspective highlights [这个观点揭示了什么].
```

### Problem/Solution 题 Body 段

```
One of the main problems is [问题].
This occurs because [原因], leading to [后果].
A possible solution would be to [措施],
which could [预期效果].
```

## 六、Task 1 图表句型模板

### 各类图表核心句型

**线图/柱图（趋势变化）：**
```
上升：rose / increased / climbed from X to Y (between 年 and 年)
下降：fell / dropped / declined from X to Y
平稳：remained stable at around X / levelled off at X
波动：fluctuated between X and Y before settling at X
顶峰：peaked at X in [year]
低谷：reached its lowest point of X in [year]
```

**饼图（比例对比）：**
```
[A] accounted for the largest share at X%, followed by [B] at Y%.
While [A] represented X%, [B] made up only Y%.
[A] and [B] together comprised X% of the total.
```

**地图（位置变化）：**
```
What was formerly [X] has been replaced by [Y].
A new [facility] has been constructed to the north/south of [X].
Where [X] once stood, there is now [Y].
The most notable change is the addition of [Y] in the [direction].
```

**流程图（被动 + 顺序词）：**
```
First, [materials] are [processed/collected/treated].
Once [X] is complete, the [materials] are then [verb-ed].
After this stage, [Y] is produced by [process].
Finally, the finished product is [stored/distributed/packaged].
```

**Overview 万能句型（所有图表必须有）：**
```
Overall, it is clear that [最显著趋势], while [对比/例外].
In general, [A] showed the most dramatic change,
whereas [B] remained relatively stable throughout the period.
```

## 九、6分核心检查清单（每次写完必查）

**Task Response：**
- [ ] 我有没有回答题目所有部分？（Two-part 题最容易漏）
- [ ] 我的立场从头到尾一致吗？
- [ ] 每个论点都有解释+例子支撑吗？

**Coherence & Cohesion：**
- [ ] 每段有一个明确的主题句吗？
- [ ] 段与段之间有逻辑衔接吗？
- [ ] 句与句用了不同的连接词？（不要全用 Furthermore）

**Lexical Resource：**
- [ ] 有没有重复用同一个词超过2次？
- [ ] 有没有话题相关的词汇（不只是通用词）？

**Grammar：**
- [ ] 没有 Comma Splice（两个完整句子用逗号连接）？
- [ ] force sb. **to do**（不是 doing）？
- [ ] cannot **be** + 过去分词（不是 cannot been）？
- [ ] 主谓一致（benefits **are**，not **is**）？
